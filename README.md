# Movie-
游客、会员
* 登录、注册
* 个人信息（修改、删除）
* 查看演员、电影信息
* 对电影评分、评价
* 观看电影

管理员
* 登录、注册
* 电影、演员信息管理
* 查看用户、删除评论
* 上映电影
